# TicTacToe


## Description 
This project displays an interactive tictactoe game which is updated by state. 

## How to Run 
Download the CSS and Javascript files and run in a React localhost browser window

## Future Improvements 
- add in a 3rd color for the + on the unchosen squares
- deactivate chosen squares so they can not be selected more than 1x per game
